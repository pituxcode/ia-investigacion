# Agentes y Herramientas del Proyecto

## Descripción del proyecto

Desarrollo de materiales para el taller "Herramientas de Inteligencia Artificial para el Fortalecimiento de la Productividad Científica" de la Universidad de Aysén.

**Fecha del taller**: 28 de enero 2026, 9:00-13:00
**Lugar**: Sala D6-Campus Lillo, Coyhaique
**Audiencia**: Cuerpo académico de la Universidad de Aysén

## Flujo de trabajo

El proyecto sigue un flujo de documentación primero:

1. **Documentación** → Se crean documentos detallados por módulo/actividad
2. **Presentación** → Se extrae la presentación desde la documentación
3. **Prompts** → Se definen los prompts para las actividades prácticas

## Estructura del proyecto

```
capacitacion-ia-investigacion/
├── AGENTS.md              # Este archivo
├── CHECKLOG.md            # Registro de cambios entre sesiones
├── descripcion.md         # Descripción general del taller
├── programa-inicial.md    # Programa oficial del taller
├── docs/                  # Documentación detallada
│   ├── modulo-1/          # Marco ético y escritura aumentada
│   └── modulo-2/          # Pulido, Inglés y Envío
├── prompts/               # Prompts para actividades prácticas
└── presentacion/          # Materiales de presentación
```

## Herramientas utilizadas

- **Claude Code**: Asistente principal para desarrollo de contenidos
- **Git**: Control de versiones
- **VS Code**: Editor de código
- **Quarto**: Sistema de documentación (flujo de trabajo del investigador)

## Convenciones

- Documentación en español chileno/latinoamericano
- Formato Markdown para todos los documentos de trabajo
- Commits descriptivos en español
